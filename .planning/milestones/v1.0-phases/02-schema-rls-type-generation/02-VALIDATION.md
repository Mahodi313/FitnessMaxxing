---

## Sampling Rate

- **After every task commit:** Run whichever of `npm run test:rls`, `npx tsc --noEmit`, or schema-grep checks is relevant to the task being committed (most tasks touch only the SQL or only the script).
- **After every plan wave:** Run the full suite (`npm run test:rls && npx tsc --noEmit && npx supabase db diff`).
- **Before `/gsd-verify-work`:** Full suite must be green; `git grep "service_role\|SERVICE_ROLE"` returns only `app/scripts/test-rls.ts`, `app/.env.example`, and docs.
- **Max feedback latency:** ~60s (full suite).

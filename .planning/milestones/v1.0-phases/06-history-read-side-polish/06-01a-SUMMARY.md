## Decisions Made

- **Three RPCs in one migration (not two)** — BLOCKER-2 fix surfaced during planning: the "Senaste 10 passen" UI list needs (session_id, weight, reps) per source session, which the day-aggregating `get_exercise_chart` cannot deliver. Shipping all three together is cheap because they share infrastructure (SECURITY INVOKER, search_path defense, revoke/grant) and Wave 0 testing them in one harness halves the round-trip.
- **`distinct on (es.session_id)` for top-sets** — Avoids window-function overhead; the inner ORDER BY collapses to the max-weight working set per session, the outer wrapper re-sorts by completed_at DESC and applies p_limit. Verified against fixture session1/session2/session3 in Wave 0 assertion K.
- **Type-cast `null as unknown as string`** — Pragmatic fix for the Supabase type-gen limitation where timestamptz RPC parameters are typed as required `string`. Runtime accepts NULL correctly (verified by all 13 + 5 assertions), and the cast is only on test-script call boundaries (never production client code).

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] Type-cast `null` parameters in RPC call sites**
- **Found during:** Final verification after Task 3 (`npx tsc --noEmit`)
- **Issue:** Generated `types/database.ts` types RPC `timestamptz` parameters as required non-nullable `string`. Both `test-exercise-chart.ts` (12 sites) and `test-rls.ts` (3 sites) call the RPCs with `null` for `p_cursor` / `p_since` (which the SQL bodies handle via `(p_cursor is null or ...)` guards). TypeScript rejected `null` → `string`. This blocked the verification gate (`tsc --noEmit` exit 0).
- **Fix:** Inserted `null as unknown as string` cast at every call site via `sed` substitution. Runtime behaviour unchanged (already verified by the harness PASSing all 13 + 5 assertions BEFORE the fix); the cast is a known-Supabase-type-gen-limitation workaround scoped to test scripts only.
- **Files modified:** `app/scripts/test-exercise-chart.ts`, `app/scripts/test-rls.ts`
- **Verification:** `npx tsc --noEmit` exits 0; `npm run test:rls` and `npm run test:exercise-chart` continue to ALL ASSERTIONS PASSED.
- **Committed in:** `0097221` (Task 3 commit; the test-exercise-chart casts were folded in because they became necessary at the final tsc gate after Task 3 landed)
